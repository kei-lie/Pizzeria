# Programmas "Picērija" izstrāde:
- [x] izveidot iespēju kā darbiniekam apkalpot pēc nejaušības izveidotus klientus;
- [x] izveidot iespēju kā darbiniekam apskatīt apkalpoto pasūtījumus un aktīvos pasūtījumus;
- [x] izveidot iespēju darbiniekam uztaisīt priekš sevis picu;

- [x] izveidot kā klientam iespēju uzsūtīt picu;
- [x] kā klientam izveidota iespēja apskatīties, cik cilvēku stāv rindā pirms viņa;
